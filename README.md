## Overview

**Easy SEO** is a professional, high-performance SEO analysis tool designed for modern web developers and site owners. It provides instant, detailed insights into your website's technical health, performance metrics, and social media readiness.

Built with a focus on speed and user experience, Easy SEO delivers actionable recommendations to improve your search engine rankings and user engagement.

## Tech Stack

*   **Frontend:** Vanilla JS, HTML5, CSS3 (Neumorphic Design)
*   **Backend:** Node.js, Express
*   **Database:** Supabase (PostgreSQL)
*   **Blockchain:** Solana (x402 Payment Protocol)
*   **Deployment:** Docker, Akash Network
